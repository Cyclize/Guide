# Bedrock Edition

**1\)** Download Bedrock edition server: [here](https://ci.nukkitx.com/job/NukkitX/job/Nukkit/job/master/lastSuccessfulBuild/artifact/target/nukkit-1.0-SNAPSHOT.jar)

**2\)** Open up your host's panel, login if you are not logged in already and select you server

[![](https://media.discordapp.net/attachments/471299720925413386/471312050023301121/firefox_2018-07-24_15-45-38.png)](https://cdn.discordapp.com/attachments/471299720925413386/471312050023301121/firefox_2018-07-24_15-45-38.png)\*\*\*\*

**3\)** Go to the `File Manager`

\`\`[![](https://media.discordapp.net/attachments/471299720925413386/471312205514539019/firefox_2018-07-23_23-28-09.png?width=223&height=300)](https://cdn.discordapp.com/attachments/471299720925413386/471312205514539019/firefox_2018-07-23_23-28-09.png)

**4\)** Select all files, Click the `Mass Actions` Button then `Delete files`

\`\`[![](https://media.discordapp.net/attachments/471299720925413386/471314544958898184/firefox_2018-07-24_15-54-07.png?width=400&height=140)](https://cdn.discordapp.com/attachments/471299720925413386/471314544958898184/firefox_2018-07-24_15-54-07.png)\*\*\*\*

**5\)** Click on the `Upload` button

[![](https://media.discordapp.net/attachments/471299720925413386/471314749972545538/firefox_2018-07-24_10-07-25.png?width=400&height=57)](https://cdn.discordapp.com/attachments/471299720925413386/471314749972545538/firefox_2018-07-24_10-07-25.png)\*\*\*\*

**6\)** Select downloaded `server.jar` and click `Open` _If there are multiple `server.jar` files \(server.jar, server\(\#\).jar, server\_\#.jar\), make sure to select latest one and when uploaded rename it to the `server.jar`_

_\`\`_[![](https://media.discordapp.net/attachments/471299720925413386/471315151740469258/firefox_2018-07-24_10-12-19.png?width=388&height=300)](https://cdn.discordapp.com/attachments/471299720925413386/471315151740469258/firefox_2018-07-24_10-12-19.png)\*\*\*\*

**7\)** When uploaded \(Blue line turns to Green line\), ...[![](https://media.discordapp.net/attachments/471299720925413386/471315340291473418/firefox_2018-07-24_10-12-38.png?width=400&height=114)](https://cdn.discordapp.com/attachments/471299720925413386/471315340291473418/firefox_2018-07-24_10-12-38.png)

... go back to the console

[![](https://media.discordapp.net/attachments/471299720925413386/471315429025906689/firefox_2018-07-24_10-19-53.png?width=219&height=300)](https://cdn.discordapp.com/attachments/471299720925413386/471315429025906689/firefox_2018-07-24_10-19-53.png)\*\*\*\*

**8\)** Start your server

[![](https://media.discordapp.net/attachments/471299720925413386/471315948280741888/firefox_2018-07-24_16-01-06.png?width=400&height=162)](https://cdn.discordapp.com/attachments/471299720925413386/471315948280741888/firefox_2018-07-24_16-01-06.png)\*\*\*\*

**9\)** Select your preferred language

[![](https://media.discordapp.net/attachments/471299720925413386/471316398879014912/firefox_2018-07-24_16-01-56.png?width=400&height=163)](https://cdn.discordapp.com/attachments/471299720925413386/471316398879014912/firefox_2018-07-24_16-01-56.png)\*\*\*\*
